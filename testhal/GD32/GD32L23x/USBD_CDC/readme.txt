*****************************************************************************
** ChibiOS/HAL - USB-CDC driver demo for GD32.                             **
*****************************************************************************

** TARGET **

The demo runs on an GD32L235R-EVAL board.

** The Demo **

The application demonstrates the use of the GD32 USBD driver.

** Build Procedure **

The demo has been tested using the free Codesourcery GCC-based toolchain
and YAGARTO.
Just modify the TRGT line in the makefile in order to use different GCC ports.
